> 需要 text generator 外掛

1. 軟體左下角進入設定，選擇第三方外掛。  
關閉安全模式之後在社群外掛市場搜尋text generator並安裝後開啟。  
![圖片](../../public/cn/obsidian1.png)  
2. 進入text generator的設定頁面。  
3. 上方API KEY一欄從[本站的Key](https://aihubmix.com/token)中產生。  
4. 下方Base Path填：
```
https://aihubmix.com/v1
```
5. 最後選擇自己要使用的模型。  
![圖片](../../public/cn/obsidian2.png)  
6. 輸入一段文字時候，點擊左側手形狀的按鈕。  
![圖片](../../public/cn/obsidian3.png)
