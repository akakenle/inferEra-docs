1. 打開 [Pot 翻譯](https://pot-app.com/)後，在服務設定中選擇添加內置服務  
![圖片](../../public/cn/pot1.png)  
2. 點擊 OpenAI  
![圖片](../../public/cn/pot2.png) 
3. 將請求地址設定為 
```
https://aihubmix.com
```    
4. API key 從[本站的 Key](https://aihubmix.com/token) 中創建生成。   
![圖片](../../public/cn/pot3.png)
5. 劃到下方保存  
![圖片](../../public/cn/pot4.png)
