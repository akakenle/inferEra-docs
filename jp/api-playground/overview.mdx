API プレイグラウンドは、リクエストを作成し、API エンドポイントをプレビューするためのインタラクティブな環境です。

<Info>
  OpenAPI を使用した API ページの自動生成により、API プレイグラウンドが自動的に生成されます。OpenAPI の使用について詳しくは[こちら](/api-playground/openapi)をご覧ください。
</Info>
