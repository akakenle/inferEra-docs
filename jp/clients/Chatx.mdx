Chatxダウンロードアドレス：https://apps.apple.com/app/id6446304087
1. 設定を開き、設定の中からAPI Server Configを見つけます。  
![画像](/media/en/chatx1.png)
![画像](/media/en/chatx2.png)
2. OpenAPIを選択します。  
![画像](/media/en/chatx3.png)
3. 上部のAPI KEY欄には[当サイトのキー](https://aihubmix.com/token)から生成したものを入力します。  
4. 下部のAPI SERVERには次のように入力します：
```
https://aihubmix.com
```
![画像](/media/en/chatx4.png)
