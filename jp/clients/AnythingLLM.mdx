## 通常の使用方法

AnythingLLM 公式ウェブサイト：[AnythingLLM](https://anythingllm.com/)  

下図の設定ページに入ります  
![画像](/media/en/any1.png)  
LLMプロバイダーはGeneric OpenAI（汎用OpenAI）を選択します。  
- Base URLの欄には以下のURLを入力します：
``` 
https://aihubmix.com/v1
```
- APIキーには[当サイトのキー](https://aihubmix.com/token)を入力します  
- Chat Model Nameの欄には、使用したいモデル名を入力します。  
- **注：Token context windowとMax Tokensは、モデルによって異なるサイズを入力できます。**  

## 非OpenAIモデルの使用方法

モデルサービスプロバイダーはGeneric OpenAIのままで、Chat Model Nameの欄に必要なモデル名を直接変更するだけです。  
ウェブサイトのモデル広場ページを開くと、使用したいモデル名をコピーできます。
