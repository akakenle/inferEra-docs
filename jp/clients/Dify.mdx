## 通常の使用状況（OpenAIモデルを使用）
ウェブページの右上隅にある自分のユーザー名をクリックして、設定画面に入ります。  
モデルプロバイダーでOpenAIを選択し、設定を開きます。  
![画像](../../public/cn/dify1.png) 
- APIキーの欄に[当サイトのキー](https://aihubmix.com/token)を入力します。  
- APIベースの欄に次のように入力します：
```
https://aihubmix.com
```  
![画像](../../public/cn/dify2.png) 

## 非OpenAIモデルの使用方法  
モデルプロバイダーのドロップダウンからOpenAI-API-compatibleを見つけ、「モデルを追加」をクリックします。  
![画像](../../public/cn/dify3.png)
- ウェブサイトの管理画面設定ページを開き、モデルリストを見つけます。
- 使用したいモデル名（例：gemini-proまたはclaude-3-opus-20240229）をコピーします。
- APIキーの欄に[当サイトのキー](https://aihubmix.com/token)を入力します。  
- APIエンドポイントURLの欄に次のように入力します：
```
https://aihubmix.com/v1
```  
![画像](../../public/cn/dify4.png)
