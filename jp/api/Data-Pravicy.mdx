## データ収集の説明

### データ保存の原則

AIHubMixは、ユーザーのプロンプトやモデルの応答内容を積極的に保存しません。

### AIHubMixが保存するデータは？

サービスの品質とシステムの安定性を確保するために、AIHubMixは各リクエストの必要なメタデータ（例えば、トークン数、遅延など）を保存します。これらの情報は、私たちのレポートやモデルのランキング、及びあなたの[使用状況](https://console.aihubmix.com/statistics)をサポートするためのみに使用されます。

このメタデータには、あなたのプロンプト内容やモデルの応答テキストは含まれず、リクエスト自体に関連する技術情報に限られます。

<Info>
  OpenAIとGrokの公式は、使用ポリシーに違反する可能性のあるコンテンツを検出した場合、アカウントの制限または禁止措置を講じ、AIHubMixにコンプライアンス通知を送信します。

  AIHubMixはプロンプトや応答内容を積極的に保存しません。アカウントが禁止され、公式のコンプライアンス通知を受け取った場合にのみ、プラットフォームは関連するリクエスト情報（リクエスト時間、プロンプト内容、モデルの応答、及び違反説明など）を受動的に受け取り、保存します。
</Info>

## ログとデータ処理

AIHubMixプラットフォームが接続している各AIプロバイダーは、独自のデータ処理方針を持っており、ログ記録メカニズムやデータ保持期間を含みます。
