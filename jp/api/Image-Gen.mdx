## インターフェースの説明

開発者がさまざまな画像生成モデルを呼び出しやすくするために、Aihubmixは統一されたグラフィックインターフェースを提供し、以下の主要なモデルをサポートしています。

- [OpenAI シリーズ](#openai)
- [Ideogram V3](#ideogram-v3)
- [Stability SD 3.5 Large](#stability)
- [Google Imagen シリーズ](#google-imagen)

**インターフェース仕様：**

```shell Curl
curl https://aihubmix.com/v1/models/<model_path>/predictions \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer sk-***" \
    -d '{
  "input": {
    "prompt": "your prompt"
  }
}'
```

**内訳：**

- `https://aihubmix.com/v1/models/` は固定プレフィックスです
- `<model_path>` はモデルパスで、以下をサポートします。
  - `opanai/gpt-image-1`
  - `opanai/dall-e-3`
  - `imagen-4.0-ultra-generate-001`
  - `imagen-4.0-generate-001`
  - `imagen-4.0-fast-generate-001`
  - `imagen-4.0-fast-generate-preview-06-06`
  - `imagen-3.0-generate-002`
  - `flux-kontext-max`
  - `flux-kontext-pro`
  - `FLUX.1-Kontext-pro`
  - `FLUX-1.1-pro`
  - `ideogram/V3`
  - `stability/Stable-Diffusion-3-5-Large`
- `sk-***` はAihubmixで生成したAPIキーです

以下は各モデルの呼び出し方法です。

## OpenAI

### サポートされているモデル：

- gpt-image-1
- dall-e-3

<Note>
  2つの逆方向モデル `gpt-4o-image-vip` と `gpt-4o-image` はサポートされていません
</Note>

### 呼び出し方法：

<CodeGroup>

```shell Curl gpt-image-1
curl https://aihubmix.com/v1/models/opanai/gpt-image-1/predictions \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer sk-***" \
    -d '{
  "input": {
    "prompt": "湖で水を飲む鹿、桜の花びらが舞い落ちる、緑と澄んだ水、日本の寺、木漏れ日、映画のような照明、広大な景色、平和",
    "size": "1024x1024", 
    "n": 1,
    "quality": "high",
    "moderation": "low",
    "background": "auto"
  }
}'
```


```shell Curl dall-e-3
curl https://aihubmix.com/v1/models/opanai/dall-e-3/predictions \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer sk-***" \
    -d '{
  "input": {
    "prompt": "湖で水を飲む鹿、桜の花びらが舞い落ちる、緑と澄んだ水、日本の寺、木漏れ日、映画のような照明、広大な景色、平和",
    "size": "1024x1024", 
    "n": 1
  }
}'
```

</CodeGroup>

## Ideogram V3

### サポートされているモデル：

- V3

<Note>
  1. V3以下のバージョン（V_2、V_1など）は古いインターフェースであり、現在サポートされていません
  2. 返されるリンクはプロキシネットワークを介してアクセスする必要があります
</Note>

### 呼び出し方法：

<CodeGroup>

```shell Curl
curl https://aihubmix.com/v1/models/ideogram/V3/predictions \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer sk-***" \
    -d '{
  "input": {

    "aspect_ratio": "2x1"

```

</CodeGroup>
