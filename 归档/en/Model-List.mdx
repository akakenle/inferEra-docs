---
title: "Model List"
description: "Retrieve available models"
icon: 'list-ul'
---

Endpoint: `GET /v1/models`

- With user log-in, retrieves available models for the user's group; without login, retrieves available models for the default group.
- If an Authorization field exists in the header, the system queries the model list configured for the corresponding token.

**Response Example:**

```json
{
  "data": [
    {
      "id": "gpt-4o-mini",
      "object": "model",
      "created": 1626777600,
      "owned_by": "OpenAI",
      "permission": [
        {
          "id": "modelperm-LwHkVFn8AcMItP432fKKDIKJ",
          "object": "model_permission",
          "created": 1626777600,
          "allow_create_engine": true,
          "allow_sampling": true,
          "allow_logprobs": true,
          "allow_search_indices": false,
          "allow_view": true,
          "allow_fine_tuning": false,
          "organization": "*",
          "group": null,

```
