端點（Endpoint）：`GET /v1/models/:model`

### 請求參數

|名稱 | 位置 | 類型 | 必選 | 說明|
|---|---|---|---|---|
|model|path|string| 是 |none|

**返回示例：**

```json
200 Response
```

```json
{
  "id": "string",
  "object": "string",
  "created": 0,
  "owned_by": "string",
  "permission": [
    {
      "id": "string",
      "object": "string",
      "created": 0,
```

|狀態碼 | 狀態碼含義 | 說明 | 資料模型|
|---|---|---|---|
|200|OK|none|Inline|
