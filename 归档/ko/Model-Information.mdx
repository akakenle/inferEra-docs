엔드포인트: `GET /v1/models/:model`

### 요청 매개변수

|이름 | 위치 | 유형 | 필수 | 설명|
|---|---|---|---|---|
|model|path|string| 예 |없음|

**응답 예시:**

```json
200 응답
```

```json

  "id": "string",
  "object": "string",
  "created": 0,
  "owned_by": "string",
  "permission": [

      "id": "string",

      "created": 0,
```
