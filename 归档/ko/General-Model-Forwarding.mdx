엔드포인트: `POST` /v1/chat/completions

**요청 본문 매개변수:**

```json
{
  "model": "gpt-4o-mini",
  "messages": [
    {
      "role": "user",
      "content": "삶의 의미는 무엇입니까?"
    }
  ]
}
```

### 요청 매개변수

|이름 | 위치 | 유형 | 필수 | 설명|
|---|---|---|---|---|
|Authorization|header|string| 아니요 |Bearer AIHUBMIX_API_KEY|
|Content-Type|header|string| 아니요 |없음|
|body|body|object| 아니요 |없음|
