## 快速配置指引：

### 1️⃣ 安裝地址

可從[官方儲存庫](https://github.com/charmbracelet/crush)獲取應用程式

### 2️⃣ 環境變數配置

啟動後填入 Aihubmix 然後輸入金鑰即可使用，
金鑰可以在 Aihubmix [「Keys」頁面](https://aihubmix.com/token) 生成。
