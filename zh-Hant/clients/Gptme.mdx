下載地址：https://github.com/ErikBjare/gptme  
專案部署後進入 ~/.config/gptme/config.toml 文件（位於用戶主目錄下）進行配置。  
配置文件例子：  
```
[env]
OPENAI_API_KEY = "sk-xxx"
MODEL = "local/gpt-4o"
OPENAI_BASE_URL = "https://aihubmix.com/v1"
```
sk-xxx 的位置換成輸入[本站的 Key](https://aihubmix.com/token)
