## 概述

歡迎使用 AIhubMix 服務。本隱私權政策旨在向您說明我們如何收集、使用、儲存和保護您的個人資訊。我們深知個人資訊對您的重要性，並會盡最大努力保護您的個人資訊安全。使用我們的服務，即表示您同意本隱私權政策中描述的資料處理程序。

## 公司資訊

AIhubMix 由註冊於美國德拉瓦州的公司營運，我們的所有伺服器和資料庫均位於美國 Microsoft Azure 雲端服務平台上。

## 資訊收集

### 我們收集的資訊

1. **帳戶資訊**：當您註冊 AIhubMix 帳戶時，我們會收集您提供的電子郵件地址、使用者名稱和密碼。
2. **使用資料**：我們會收集您使用我們服務的相關資訊，包括：
   - API 調用記錄
   - 模型使用情況
   - Token 消耗量
   - IP 地址和設備資訊
   - 與模型請求相關的 Web 應用程式 URL
3. **支付資訊**：如果您購買我們的服務，我們會收集必要的支付資訊，但完整的支付卡詳細資料由我們的支付處理合作夥伴安全處理。
4. **錢包地址**：如果您選擇使用加密貨幣支付，我們會收集您的錢包地址。

### 資訊收集方式

1. **直接提供**：您在註冊、使用或購買我們服務時直接提供的資訊。
2. **自動收集**：透過 cookies、日誌文件和其他技術自動收集的資訊。
3. **第三方來源**：在某些情況下，我們可能從第三方服務提供商獲取資訊。

## 資訊使用

我們使用收集的資訊用於：

1. **提供服務**：處理 API 請求，提供大模型 (LLM) 訪問服務。
2. **帳戶管理**：建立和管理您的帳戶，驗證身份。
3. **計費和結算**：處理支付，發送帳單，管理訂閱。
4. **服務改進**：分析使用模式，優化我們的服務和 API 效能。
5. **客戶支援**：回應您的問題和請求。
6. **安全保障**：偵測和防止欺詐、濫用和安全威脅。
7. **通訊**：發送服務通知、更新和行銷資訊（如您同意）。

## 隱私權聲明

我們高度重視使用者隱私，致力於保護使用者的個人資訊。我們不會以任何方式收集、記錄或儲存使用者輸入的任何文字或由 OpenAI 伺服器返回的任何文字。我們不會向 OpenAI 或任何第三方提供有關 API 調用者的身份的任何資訊，包括但不限於 IP 地址和使用者代理字串。

## 資訊共享

我們不會出售您的個人資訊。我們可能在以下情況共享您的資訊：

1. **服務提供商**：與幫助我們提供服務的第三方服務提供商共享（如支付處理商、雲服務提供商、分析服務如 Google Analytics）。
2. **合規要求**：當法律要求或為回應合法請求（如法院命令或傳票）時。
3. **保護權利**：當我們認為披露是保護 AIhubMix、我們的使用者或公眾的權利和安全所必需的。
4. **業務轉讓**：如果 AIhubMix 參與合併、收購或資產出售，您的資訊可能作為轉讓資產的一部分被轉移。

這些第三方僅在履行其職能所需的必要情況下才能訪問您的資訊，不得將其用於其他目的。

## 資料安全

我們實施了適當的技術和組織措施來保護您的個人資訊，包括：

1. **加密傳輸**：所有 API 通訊透過 TLS/SSL 加密。
2. **安全儲存**：敏感資料使用行業標準加密儲存。
3. **訪問控制**：嚴格限制員工對個人資料的訪問。
4. **定期審計**：定期評估和更新我們的安全措施。

儘管我們努力保護您的個人資訊，但請注意，沒有任何安全措施是完全無懈可擊的。

## 資料保留

我們會在實現本隱私權政策中描述的目的所需的時間內保留您的個人資訊，除非法律要求或允許更長的保留期。當不再需要您的個人資訊時，我們將安全地刪除或匿名化這些資訊。

## 資料儲存和國際傳輸

我們的伺服器和資料庫位於美國的 Microsoft Azure 雲端服務平台上。如果您位於美國以外的地區，請注意您的資訊將被傳輸到美國並在那裡處理。美國的資料保護法律可能與您所在國家/地區的法律不同。透過使用我們的服務，您同意將您的資訊傳輸到美國並在那裡處理。

## API 調用內容

1. **使用者輸入**：透過我們 API 發送到大模型 (LLM) 的提示和查詢內容。
2. **模型輸出**：大模型 (LLM) 生成的回應內容。

### 關於 API 調用內容的特別說明

- 我們不會將您透過 API 發送的內容用於訓練模型。
- 我們不會主動審查 API 調用內容。
- AI 模型提供商通常會暫時儲存提示資料或根本不儲存，但如果他們可以匿名收集提示，則少數提供商會提供較低的價格。要查看特定模型提供商的資料政策，請訪問其模型頁面。

## 您的權利和選擇

您可以透過我們平台的使用者介面更新可修改的個人資訊。如果您希望請求刪除您的帳戶和個人資料，請訪問您的設定頁面或透過下方聯繫資訊與我們聯繫。

## Cookie 政策

我們使用 cookies 和類似技術來提供、保護和改進我們的服務。cookies 是儲存在您設備上的小文字文件，用於記住您的偏好和其他資訊。使用我們的服務即表示您同意我們使用這些方法。您可以透過瀏覽器設定控制和刪除 cookies。但請注意，禁用 cookies 可能會影響我們網站的某些功能。

## 兒童隱私

我們的服務不面向 13 歲以下的兒童。我們不會故意收集 13 歲以下兒童的個人資訊。如果您發現我們可能收集了 13 歲以下兒童的個人資訊，請立即聯繫我們，我們將採取措施刪除這些資訊。

## 隱私權政策更新

我們可能會不時更新本隱私權政策，恕不另行通知。當我們進行重大更改時，我們會在網站上發布通知，並更新政策頂部的「最後更新」日期。在發布更改後，您繼續使用我們的服務即表示您接受並同意這些更改。我們鼓勵您定期查看本隱私權政策，以了解我們如何保護您的資訊。

## 適用法律

本隱私權政策受美國德拉瓦州法律管轄，不考慮法律衝突原則。與本隱私權政策相關的任何爭議應提交給德拉瓦州法院專屬管轄。

## 終止

我們可隨時以任何理由（包括違反服務條款）終止您對服務的使用，並向您發出通知。您可以透過停止使用我們的服務隨時終止您對服務的使用。

終止後，除本質上應繼續存在的權利和義務外，所有權利和義務均將消失，包括但不限於所有權條款、免責聲明、責任限制和爭議解決條款。
