## 快速上手

1. **儲值**  
   - 進入[「儲值」頁面](https://aihubmix.com/topup)完成儲值，確保帳戶餘額 > 0  
2. **創建 API Key**  
   - 打開[「Keys」頁面](https://aihubmix.com/token)，創建並複製自己的 Key，新手建議只填 `Key 名稱`
3. **獲取中轉資訊**  
   - 穩定中轉接口地址：`https://aihubmix.com`  
   - 中轉 API Key（不要硬編碼在專案中）：`sk-***` 
   - 備用伺服器地址（新功能測試專用）：`https://api.aihubmix.com`  
4. **替換 API 地址**  
   - 如果您是開發者，請參考 [OpenAI 官方文件](https://platform.openai.com)  
   - 開發時，將文件中的 `https://api.openai.com` 替換為 `https://aihubmix.com` 即可  
   - Claude 模型也支援原生 `v1/messages` 接口，詳情請參考 [Claude 原生接口調用](/cn/api/Claude-Native)

<Tip>
**提示**：
在一些第三方軟體或平台輸入自定義 URL 時，可能需要追加 `/v1` 或 `/v1/chat/completions` 
</Tip>
如：
- `https://aihubmix.com`
- `https://aihubmix.com/v1`（目前最常見）
- `https://aihubmix.com/v1/chat/completions`
