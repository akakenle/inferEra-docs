<Note>
- 400 상태 코드는 일반적으로 매개변수 오류를 나타냅니다. API 문서를 확인하세요. 대부분의 400 오류는 업스트림 통과 오류입니다.
- 오류 응답의 응답 ID가 오류 추적 및 문제 해결 목적으로 tid(traceId)로 변경되었습니다.
</Note>

| HTTP 상태 코드 | 오류 식별자 | 오류 메시지 | 일반적인 원인 |
|------------------|------------------|---------------|--------------|
| 503 | - | 잘못된 모델 ID입니다. 모델 페이지를 보기 위해 요청하거나 이 모델을 사용할 권한이 없습니다 | 요청을 처리할 수 있는 채널이 없음 |
| 503 | - | 제공업체에 의해 속도 제한됨 – 더 높은 동시 처리를 위해 지원팀에 문의하거나 나중에 다시 시도하세요. | 모델이 공식 속도 제한에 걸림 |
| 429 | - | xx 모델이 너무 많은 요청을 받았습니다. 나중에 다시 시도하세요. | 요청 빈도가 제한을 초과함 |
| 403 | insufficient_user_quota | 계정 잔액이 부족합니다. API를 계속 사용하려면 계정을 충전하세요. | 사용자 잔액 부족, 충전 필요 |
| 403 | - | 계정이 정지되었습니다. | 사용자 상태 비활성화 또는 블랙리스트 |
| 403 | - | 금지됨 – 권한이 부족합니다. | 사용자 역할 권한 부족 |
| 403 | - | 금지됨 – 키(요청 키의 마지막 6자리)는 승인된 IP 범위에서만 허용됩니다. | IP가 토큰의 허용된 네트워크 범위에 없음 |
| 403 | - | 금지됨 – 키(요청 키의 마지막 6자리)가 요청된 모델에 액세스할 권한이 없습니다. | 토큰이 요청된 모델을 지원하지 않음 |
| 403 | - | 키 오류；(요청 키의 마지막 6자리) | 관리자가 아닌 사용자가 채널을 지정하려고 시도함 |
| 403 | - | 금지됨 – 채널이 비활성화되었습니다. | 채널 상태가 비활성화됨 |
| 401 | - | 인증되지 않음 – 액세스 토큰이 제공되지 않았습니다; 로그인하고 유효한 토큰을 포함하세요. | Authorization 헤더가 제공되지 않음 |
| 401 | - | 인증되지 않음 – 액세스 토큰이 유효하지 않거나 만료되었습니다. | 액세스 토큰 검증 실패 |
| 400 | - | 잘못된 요청 – 유효하지 않은 채널 ID입니다. | 채널 ID 형식 오류 또는 존재하지 않음 |
| 400 | prompt_missing | 프롬프트가 필요합니다 | 이미지 생성용 프롬프트 누락 |
| 400 | prompt_too_long | 프롬프트가 너무 깁니다 | 프롬프트가 길이 제한을 초과함 |
| 400 | text_too_long | 입력이 너무 깁니다 | 오디오 입력 텍스트가 너무 김 |
| 400 | size_not_supported | 크기가 지원되지 않습니다 | 이미지 크기가 모델에서 지원되지 않음 |
| 400 | n_not_within_range | n의 값이 유효하지 않습니다 | n 매개변수 값이 유효한 범위에 없음 |
