## 데이터 수집 설명

### 데이터 저장 원칙

AIHubMix는 귀하의 프롬프트(Prompt)나 모델 응답 내용을 적극적으로 저장하지 않습니다.

### AIHubMix가 저장하는 데이터는 무엇인가요?

서비스 품질과 시스템 안정성을 보장하기 위해, AIHubMix는 매 요청의 필수 메타데이터(예: 토큰 수, 지연 시간 등)를 저장합니다. 이 정보는 우리의 보고서 및 모델 순위를 지원하고 귀하의 [사용량 상황](https://console.aihubmix.com/statistics)에만 사용됩니다.

이 메타데이터는 귀하의 프롬프트 내용이나 모델 응답 텍스트를 포함하지 않으며, 요청 자체와 관련된 기술 정보에 한정됩니다.

<Info>
  OpenAI와 Grok 공식은 사용 정책을 위반할 가능성이 있는 내용을 감지할 경우 계정 제한 또는 정지 조치를 취하고 AIHubMix에 준수 알림을 전송합니다.

  AIHubMix는 프롬프트나 응답 내용을 적극적으로 저장하지 않습니다. 계정 정지가 발생하고 공식 준수 알림을 받은 경우에만 플랫폼은 관련 요청 정보(요청 시간, 프롬프트 내용, 모델 응답 및 위반 설명 등)를 수동으로 수신하고 보관합니다.
</Info>
