## 빠른 설정 가이드:

### 1️⃣ 설치 주소

[공식 저장소](https://github.com/charmbracelet/crush)에서 애플리케이션을 다운로드할 수 있습니다

### 2️⃣ 환경 변수 설정

시작 후 Aihubmix를 입력하고 API 키를 입력하면 사용할 수 있습니다.
API 키는 Aihubmix [「Keys」페이지](https://aihubmix.com/token)에서 생성할 수 있습니다.
