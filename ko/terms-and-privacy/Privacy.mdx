## 개요

AIhubMix 서비스에 오신 것을 환영합니다. 본 개인정보처리방침은 당사가 귀하의 개인정보를 수집, 사용, 저장 및 보호하는 방법을 알려드리기 위함입니다. 당사는 귀하에게 개인정보의 중요성을 이해하고 있으며, 귀하의 개인정보 보안을 보호하기 위해 최선을 다할 것입니다. 당사 서비스를 이용함으로써 귀하는 본 개인정보처리방침에 설명된 데이터 처리 절차에 동의하는 것입니다.

## 회사 정보

AIhubMix는 미국 델라웨어 주에 등록된 회사에 의해 운영되며, 당사의 모든 서버와 데이터베이스는 미국 Microsoft Azure 클라우드 서비스 플랫폼에 위치하고 있습니다.

## 정보 수집

### 당사가 수집하는 정보

1. **계정 정보**: AIhubMix 계정을 등록할 때 귀하가 제공하는 이메일 주소, 사용자 이름 및 비밀번호를 수집합니다.
2. **사용 데이터**: 당사는 다음을 포함하여 당사 서비스 사용과 관련된 정보를 수집합니다:
   - API 호출 기록
   - 모델 사용량
   - 토큰 소비량
   - IP 주소 및 장치 정보
   - 모델 요청과 관련된 웹 애플리케이션 URL
3. **결제 정보**: 당사 서비스를 구매하는 경우, 당사는 필요한 결제 정보를 수집하지만, 전체 결제 카드 세부 정보는 당사의 결제 처리 파트너에 의해 안전하게 처리됩니다.
4. **지갑 주소**: 암호화폐로 결제하기로 선택한 경우, 당사는 귀하의 지갑 주소를 수집합니다.

### 정보 수집 방법

1. **직접 제공**: 서비스 등록, 사용 또는 구매 시 귀하가 직접 제공하는 정보.
2. **자동 수집**: 쿠키, 로그 파일 및 기타 기술을 통해 자동으로 수집되는 정보.
3. **제3자 출처**: 경우에 따라 제3자 서비스 제공업체로부터 정보를 얻을 수 있습니다.

## 정보 사용

당사는 수집된 정보를 다음 목적으로 사용합니다:

1. **서비스 제공**: API 요청 처리 및 대규모 모델(LLM)에 대한 액세스 제공.
2. **계정 관리**: 계정 생성 및 관리, 신원 확인.
3. **청구 및 정산**: 결제 처리, 청구서 발송, 구독 관리.
4. **서비스 개선**: 사용 패턴 분석, 서비스 및 API 성능 최적화.
5. **고객 지원**: 귀하의 질문 및 요청에 응답.
6. **보안 보장**: 사기, 남용 및 보안 위협 감지 및 방지.
7. **커뮤니케이션**: 서비스 알림, 업데이트 및 마케팅 정보 발송 (동의하는 경우).

# 개인정보 보호 선언

본 프로젝트는 개인정보 보호를 최우선으로 하며, 사용자 개인정보 보호에 전념합니다. 본 프로젝트는 사용자가 입력한 텍스트 또는 OpenAI 서버에서 반환된 텍스트를 어떤 방식으로든 수집, 기록 또는 저장하지 않습니다. 본 프로젝트는 IP 주소 및 사용자 에이전트 문자열을 포함하되 이에 국한되지 않는 API 호출자의 신원에 대한 어떠한 정보도 OpenAI 또는 제3자에게 제공하지 않습니다.

## 정보 공유

당사는 귀하의 개인정보를 판매하지 않습니다. 당사는 다음 상황에서 귀하의 정보를 공유할 수 있습니다:

1. **서비스 제공업체**: 서비스 제공을 돕는 제3자 서비스 제공업체(예: 결제 처리업체, 클라우드 서비스 제공업체, Google Analytics와 같은 분석 서비스)와 공유.
2. **규정 준수 요구사항**: 법률에 의해 요구되거나 합법적인 요청(예: 법원 명령 또는 소환장)에 응답해야 하는 경우.
3. **권리 보호**: AIhubMix, 당사 사용자 또는 대중의 권리와 안전을 보호하기 위해 공개가 필요하다고 판단되는 경우.
4. **사업 양도**: AIhubMix가 합병, 인수 또는 자산 매각에 관여하는 경우, 귀하의 정보는 양도된 자산의 일부로 이전될 수 있습니다.

이러한 제3자는 자신의 기능을 수행하는 데 필요한 경우에만 귀하의 정보에 액세스할 수 있으며, 다른 목적으로 사용할 수 없습니다.

## 데이터 보안

당사는 귀하의 개인정보를 보호하기 위해 다음과 같은 적절한 기술적 및 조직적 조치를 구현했습니다:

1. **암호화된 전송**: 모든 API 통신은 TLS/SSL을 통해 암호화됩니다.
2. **보안 저장**: 민감한 데이터는 산업 표준 암호화를 사용하여 저장됩니다.
3. **접근 제어**: 직원의 개인 데이터 접근을 엄격히 제한합니다.
4. **정기 감사**: 보안 조치를 정기적으로 평가하고 업데이트합니다.

당사는 귀하의 개인정보를 보호하기 위해 노력하지만, 어떠한 보안 조치도 완전히 완벽하지 않다는 점을 유의하십시오.

## 데이터 보존

당사는 법률에 의해 더 긴 보존 기간이 요구되거나 허용되지 않는 한, 본 개인정보처리방침에 설명된 목적을 달성하는 데 필요한 기간 동안 귀하의 개인정보를 보존합니다. 귀하의 개인정보가 더 이상 필요하지 않을 경우, 당사는 이를 안전하게 삭제하거나 익명화합니다.

## 데이터 저장 및 국제 전송

당사의 서버와 데이터베이스는 미국 Microsoft Azure 클라우드 서비스 플랫폼에 위치하고 있습니다. 귀하가 미국 외부에 거주하는 경우, 귀하의 정보가 미국으로 전송되어 처리된다는 점을 유의하십시오. 미국의 데이터 보호 법률은 귀하의 국가와 다를 수 있습니다. 당사 서비스를 이용함으로써 귀하는 귀하의 정보가 미국으로 전송되어 처리되는 것에 동의하는 것입니다.

## API 호출 내용

1. **사용자 입력**: 당사 API를 통해 대규모 모델(LLM)로 전송되는 프롬프트 및 쿼리 내용.
2. **모델 출력**: 대규모 모델(LLM)에 의해 생성된 응답.

### API 호출 내용에 대한 특별 참고 사항

- 당사는 귀하가 API를 통해 전송하는 내용을 모델 훈련에 사용하지 않습니다.
- 당사는 API 호출 내용을 적극적으로 검토하지 않습니다.
- AI 모델 제공업체는 일반적으로 프롬프트 데이터를 일시적으로 저장하거나 전혀 저장하지 않지만, 익명으로 프롬프트를 수집할 수 있다면 일부 제공업체는 더 낮은 가격을 제공할 수 있습니다. 특정 모델 제공업체의 데이터 정책을 보려면 해당 모델 페이지를 방문하십시오.

## 귀하의 권리 및 선택

귀하는 당사 플랫폼의 사용자 인터페이스를 통해 수정 가능한 개인정보를 업데이트할 수 있습니다. 계정 및 개인 데이터 삭제를 요청하려면 설정 페이지를 방문하거나 아래 연락처 정보를 사용하여 당사에 문의하십시오.

## 쿠키 정책

당사는 서비스를 제공, 보호 및 개선하기 위해 쿠키 및 유사 기술을 사용합니다. 쿠키는 귀하의 기본 설정 및 기타 정보를 기억하기 위해 귀하의 장치에 저장되는 작은 텍스트 파일입니다. 당사 서비스를 이용함으로써 귀하는 이러한 방법의 사용에 동의하는 것입니다. 귀하는 브라우저 설정을 통해 쿠키를 제어하고 삭제할 수 있습니다. 그러나 쿠키를 비활성화하면 당사 웹사이트의 특정 기능에 영향을 미칠 수 있다는 점을 유의하십시오.
