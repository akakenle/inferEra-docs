## 1. 약관 동의

AIhubMix(이하 "당사", "본 서비스" 또는 "AIhubMix")에 오신 것을 환영합니다. 본 서비스 약관("약관")은 귀하의 당사 서비스 사용과 관련하여 귀하와 AIhubMix 간의 법적 계약을 구성합니다. 당사 웹사이트, API 또는 관련 서비스에 액세스하거나 사용함으로써 귀하는 본 약관을 읽고 이해했으며 이에 구속되는 데 동의함을 확인합니다. 본 약관에 동의하지 않는 경우, 당사 서비스를 사용하지 마십시오.

## 2. 서비스 설명

AIhubMix는 OpenAI, Claude, Gemini 등을 포함하되 이에 국한되지 않는 다양한 AI 모델에 대한 액세스를 제공하는 대규모 모델(LLM) API 통합 플랫폼입니다. 당사 서비스를 통해 사용자는 통합 API 인터페이스를 통해 이러한 모델을 호출하고 실시간 온라인 검색과 같은 추가 기능을 제공할 수 있습니다. AIhubMix는 현재 테스트 단계에 있습니다. 당사는 특정 모델의 가용성을 포함하여 서비스의 어떤 부분이라도 언제든지 변경하거나 중단할 수 있습니다. "실험적"으로 표시된 일부 모델은 충분한 사용량을 확보하고 운영 경제성을 충족하는지 여부를 평가하기 위해 평가 중이며, 언제든지 제거될 수 있습니다.

비실험적 모델 제거의 경우, 당사는 다음 조치를 취할 것입니다:
- 공식 커뮤니티 채널에 알림 제공
- 일정 기간 동안 해당 모델에 대한 요청을 유사한 가격 및 기능을 가진 다른 모델로 재라우팅
- 최소 2주의 유예 기간 후, 해당 모델에 대한 요청 재라우팅을 중단하고 대신 404 오류 반환

## 3. 사용자 자격 및 계정 보안

### 3.1 연령 요건

13세 미만의 모든 사람은 AIhubMix 사용이 금지됩니다. 귀하는 거주 관할권에서 법적 성년 연령 이상임을 진술하며, 법적 성년이 아닌 경우 귀하의 부모 또는 법적 보호자가 본 약관에 동의하고 귀하를 대신하여 본 약관을 수락하고 귀하의 사용에 대한 책임을 진다는 것을 확인해야 합니다.

### 3.2 등록 요건

당사 서비스를 사용하려면 계정을 생성해야 할 수 있습니다. 귀하는 정확하고 완전하며 최신 등록 정보를 제공하는 데 동의하며, 해당 정보의 정확성과 완전성을 유지할 책임이 있습니다.

### 3.3 계정 보안

귀하는 비밀번호 보안을 포함하여 계정의 기밀을 유지할 책임이 있으며, 귀하의 계정에서 발생하는 모든 활동에 대해 전적으로 책임이 있습니다. 귀하는 귀하의 계정에 대한 무단 사용 또는 기타 보안 위반이 발생할 경우 즉시 당사에 통지하는 데 동의합니다.

### 3.4 API 키 보안

API 키를 안전하게 보관해야 합니다. API 키가 유출되면 무단 사용 및 추가 요금이 발생할 수 있습니다. 당사는 API 키 유출로 인해 발생하는 손실이나 손해에 대해 책임지지 않습니다.

## 4. 서비스 사용 규칙

### 4.1 합법적인 사용

귀하는 당사 서비스를 합법적인 목적으로만 사용하고 모든 해당 법률 및 규정을 준수하는 데 동의합니다. 귀하는 불법적이거나 금지된 활동에 참여하기 위해 당사 서비스를 사용해서는 안 됩니다.

### 4.2 금지된 행위

귀하는 다음을 해서는 안 됩니다:
- 불법적이거나 유해하거나 위협적이거나 학대적이거나 괴롭히거나 명예를 훼손하거나 음란하거나 기타 불쾌한 콘텐츠를 생성, 업로드 또는 배포하기 위해 당사 서비스를 사용
- 타인의 권리 또는 해당 법률을 침해
- 당사 모델, 알고리즘 및 시스템의 소스 코드 또는 기본 구성 요소를 역설계, 디컴파일, 디스어셈블, 번역 또는 기타 방식으로 발견하려고 시도
- 당사 서비스, 모델 또는 기본 기술을 남용, 손상, 방해, 역설계 또는 중단
- 사용 제한을 우회하거나 공개되지 않은 서비스 기능에 액세스하려고 시도
- 서비스 또는 서비스에 연결된 서버 또는 네트워크의 정상적인 작동을 방해하거나 중단
- 서비스에 불합리하거나 불균형적인 부담을 줄 수 있는 활동에 참여
- 당사가 명시적으로 허용하지 않는 한 자동화된 방법을 사용하여 서비스에서 정보를 수집
- 당사의 명시적인 서면 허가 없이 당사 API를 재판매, 임대 또는 기타 방식으로 상업화
- AIhubMix에서 생성된 콘텐츠가 사람이 생성한 것이라고 진술

### 4.3 콘텐츠 책임

귀하는 당사 API를 통해 전송된 모든 자료와 모델에서 수신된 모든 결과(총칭하여 "귀하의 콘텐츠")에 대해 전적으로 책임이 있습니다. 귀하는 귀하의 콘텐츠에 대한 모든 권리를 보유합니다. 당사는 귀하가 당사 서비스를 사용하여 생성하거나 처리하는 콘텐츠를 모니터링하거나 제어하지 않으며, 해당 콘텐츠에 대해 책임지지 않습니다.

## 5. 요금 및 결제

### 5.1 가격

당사 서비스에는 요금이 부과될 수 있습니다. 현재 가격 및 청구 조건은 당사 웹사이트에 게시되어 있으며 수시로 변경될 수 있습니다. 모든 요금은 달리 명시되지 않는 한 미국 달러로 표시됩니다.

### 5.2 청구 방법

당사는 API 호출량, 토큰 소비량 또는 기타 사용량 측정 기준에 따라 요금을 청구합니다. 특정 청구 방법은 귀하가 선택한 서비스 계획 및 사용하는 모델에 따라 다릅니다.

### 5.3 결제 방법

사용자는 결제 카드 또는 암호화폐 지갑을 사용하여 모델 요금을 지불하기 위해 크레딧을 구매할 수 있습니다. 크레딧은 환불되지 않으며 모델 요금을 지불하는 데 사용할 수 있으며 언제든지 재충전할 수 있습니다. 크레딧은 구매 후 12개월 후에 만료될 수 있습니다.

당사는 결제를 안전하게 처리하기 위해 제3자 결제 처리업체를 사용합니다. 암호화폐가 아닌 결제에는 추가 처리 수수료가 발생할 수 있습니다. 귀하의 결제와 관련하여 사기 분쟁 또는 사기와 관련된 환불을 받는 경우, 귀하는 해당 크레딧 금액과 분쟁 처리 비용을 잃게 됩니다.

개발자는 모델 요금을 직접 지불하거나 인증 프로세스를 통해 사용자가 지불하도록 허용할 수 있습니다. 귀하의 자체 공급자 API 키를 사용하는 경우(통합을 통해), 당사는 귀하의 계정에서 해당 서비스 요금을 공제할 것입니다.

### 5.4 환불 정책

모든 요금은 해당 법률에서 요구하는 경우를 제외하고는 환불되지 않습니다. 선불 사용자의 경우, 계정 폐쇄 후 미사용 잔액은 당시 당사의 정책에 따라 환불될 수 있습니다.

### 6.1 당사의 지적 재산권

서비스 및 그 원본 콘텐츠, 기능 및 디자인은 지적 재산권 법률에 의해 보호됩니다. 당사 서비스를 사용한다고 해서 사용자 또는 개발자에게 지적 재산권이 이전되는 것은 아닙니다. AIhubMix 및 그 라이선스 제공자는 서비스에 대한 모든 권리, 소유권 및 이익을 보유합니다.

### 6.3 피드백

귀하가 당사 서비스에 대한 제안, 아이디어 또는 피드백을 제공하는 경우, 귀하는 당사가 귀하에게 어떠한 보상도 없이 해당 정보를 제한 없이 사용할 수 있음에 동의합니다.
