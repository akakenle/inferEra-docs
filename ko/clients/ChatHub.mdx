1. ChatHub 공식 웹사이트에서 Chrome 확장 프로그램을 설치합니다: https://chathub.gg  

2. ChatHub 확장 프로그램의 왼쪽 하단에 있는 톱니바퀴 아이콘을 클릭하여 설정에 액세스합니다 (이미지에서 빨간색 1로 표시).  

3. "API 키를 통해 사용" 목록에서 OpenAI의 "구성" 버튼을 클릭합니다 (이미지에서 빨간색 2로 표시).  

4. [저희 사이트의 키](https://aihubmix.com/token)에서 생성할 수 있는 AiHubMix API 키(키 형식: sk-xxxxxxxx)를 입력한 다음 복사하여 붙여넣습니다 (이미지에서 빨간색 3으로 표시).  

5. API 호스트를 AiHubMix 주소로 변경합니다 (이미지에서 빨간색 4로 표시):  
```
https://aihubmix.com
```
