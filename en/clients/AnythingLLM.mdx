## Common Usage Method

AnythingLLM Official Website: [AnythingLLM](https://anythingllm.com/)  

Enter the settings page as shown in the image below    
![Image](../../public/cn/any1.png)
Select Generic OpenAI for the LLM provider  
- In the Base URL field, enter the following URL:
```
https://aihubmix.com/v1
```
- In the APIkey field, enter [the Key from this site](https://aihubmix.com/token)  
- In the Chat Model Name field, enter the model name you need.  
- **Note: Token context window and Max Tokens can vary in size depending on the model.**  

## Non-OpenAI Model Usage Method

Keep Generic OpenAI selected for the model provider, and manually modify the desired model name in the Chat Model Name field.  
You can copy the model name you want to use from the model square page on the website.
