---
title: "Using GCMP"
---

1. Search for `GCMP` in the VS Code Extensions Marketplace and install it.

   ![](public/en/GCMP-1.png)

2. Open the **GitHub Copilot Chat** panel in VS Code, then scroll to the bottom of the model selector and choose **Model Management**.

   ![](public/en/GCMP-2.png)

3. Click **Manage Models**, select the provider **OpenAI / Anthropic Compatible**, and create a new model.

   ![](public/en/GCMP-3.png)

4. Fill in the basic information and select **aihubmix** as the provider.

   ![](public/en/GCMP-4.png)

5. Manage your API keys and enter your **AIHubMix API Key**.

   ![](public/en/GCMP-5.png)
