### Configuration Guide

1. Enter the workspace, add a **Chat Model** node, and create new credentials.

![n8n](/public/en/n8n-1.jpg)

2. Enter a name, the API Key, and the Base URL, then save.

![n8n](/public/en/n8n-2.png)

3. Select the newly created credentials, add the model you want to use, and disable **`Use Response API`**.

![n8n](/public/en/n8n-3.png)

4. Open **`Open Chat`** to start using AIHubMix models in N8n.

![n8n](/public/en/n8n-4.jpg)

![n8n](/public/en/n8n-5.png)
