## Common usage method
Lobe-Chat official website: [chat-preview.lobehub.com](https://chat-preview.lobehub.com/?utm_source=aihubmix&utm_medium=website&utm_campaign=references)  

Click to enter settings as shown below  
![Image](../../public/cn/LC1.png)
![Image](../../public/cn/LC2.png)  
- API Key: Enter [the Key from this site](https://aihubmix.com/token)  
- Proxy Address: Enter the following URL directly:
``` 
https://aihubmix.com/v1
```
(It is recommended to enable "Use Client Request Mode")  
![Image](../../public/cn/LC3.png)  
Finally, add the model you want to use in the model list  
![Image](../../public/cn/LC4.png)

## Non-openai model usage method

The choice of model service provider remains openai, and you can manually add the required model name in the model list.  
You can copy the model name you want to use by opening the model square page on the website.
