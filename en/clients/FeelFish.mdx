### 官网

FeelFish Website: https://www.feelfish.com

### 配置教程

1. Click the Settings icon in the top-right corner to open the settings window.

![Fs 1 Pn](/public/en/fs-1.png)

2. Switch to the “Model Configuration” tab. Scroll down to the Advanced Settings section and click Add Model API.

![Fs 1 Pn](/public/en/fs-1.png)

3. Fill in the required information in the popup window. FeelFish already includes the preset AIHubMix API address, so all you need to do is enter a random ID and the API key generated on the AIHubMix platform, then click Save to complete the setup.

![Fs 1 Pn](/public/en/fs-1.png)

4. Go back to the top of the configuration panel and add available models.

![Fs 1 Pn](/public/en/fs-1.png)

5. Import models from AIHubMix into FeelFish.

![Fs 1 Pn](/public/en/fs-1.png)

6. When writing, simply select the desired model to start using it right away.
   
![Fs 1 Pn](/public/en/fs-1.png)
