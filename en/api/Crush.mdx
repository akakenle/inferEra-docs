## Quick Setup Guide:

### 1️⃣ Installation

Get the application from the [official repository](https://github.com/charmbracelet/crush)

### 2️⃣ Environment Configuration

After launching, fill in Aihubmix and enter your API key to use it.
API keys can be generated on the Aihubmix [「Keys」page](https://aihubmix.com/token).
