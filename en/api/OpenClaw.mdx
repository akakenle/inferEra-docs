## Installation

<Note>
  Please ensure Node >= 22 before installation.
</Note>

Run the installation command

```shellscript
npm install -g openclaw@latest
```

## Configure AIHubMix

### Method 1: Quick Setup via Plugin (Recommended)

<Info>
  Plugin source code: [aihubmix-auth](https://github.com/akakenle/aihubmix-auth)
</Info>

Install the AIHubMix auth plugin and restart the Gateway:

```shellscript
openclaw plugins install @akaknele/aihubmix-auth
openclaw gateway restart
```

Run the auth command. Enter your AIHubMix API Key when prompted. The plugin will automatically sync available models and set AIHubMix as the default provider:

```shellscript
openclaw models auth login --provider aihubmix --method api-key --set-default
```

The plugin organizes models into the following groups:
- **openai** — OpenAI models
- **anthropic** — Anthropic models
- **google** — Google models
- **other** — Other models

Once configured, jump to the [Start](#start) section to begin using it.

### Method 2: Manual Configuration

Open the `~/.openclaw/openclaw.json` file and add the following configuration:

```json
{
  "models": {
    "mode": "merge",
    "providers": {
      "aihubmix": {
        "baseUrl": "http://aihubmix.com/v1",
        "apiKey": "AIHUBMIX_API_KEY",
        "api": "openai-completions",
        "models": [
          {
            "id": "claude-sonnet-4-5",
            "name": "claude-sonnet-4-5",
            "reasoning": false,
            "input": [
              "text"
            ],
            "cost": {
              "input": 0,
              "output": 0,
              "cacheRead": 0,
              "cacheWrite": 0
            },
            "contextWindow": 1000000,
            "maxTokens": 64000
          }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "aihubmix/claude-sonnet-4-5"
      },
      "workspace": "your_workspace_path",
      "compaction": {
        "mode": "safeguard"
      },
      "maxConcurrent": 4,
      "subagents": {
        "maxConcurrent": 8
      }
    }
  }
}
```
