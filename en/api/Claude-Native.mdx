Endpoint: `POST /v1/messages`

**Request Body Parameters:**

```json

  "model": "claude-3-5-sonnet-20241022",
  "max_tokens": 1024,
  "messages": [
    {
      "role": "user",
      "content": "What is the meaning of life?"

}
```
