## Configuration

1. Run VS Code and install Kilo Code.

<Frame>
  ![I Shot 2026 03 25 15 40 03 1](/images/iShot_2026-03-25_15.40.03-1.png)
</Frame>

2. Click the `Kilo Code` plugin icon to enter the configuration page.

<Frame>
  ![微信图片 2026 03 25 154246 867](/images/微信图片_2026-03-25_154246_867.png)
</Frame>

3. Set the API Provider to `AIhubmix`.

<Frame>
  ![I Shot 2026 03 25 15 44 58](/images/iShot_2026-03-25_15.44.58.png)
</Frame>

4. Enter the API Key and select the model you wish to use.

<Frame>
  ![I Shot 2026 03 25 15 46 39](/images/iShot_2026-03-25_15.46.39.png)
</Frame>

5. Save the information.

<Frame>
  ![I Shot 2026 03 25 15 50 20](/images/iShot_2026-03-25_15.50.20.png)
</Frame>
