端点（Endpoint）： `GET /v1/models`

- 有用户登录获取用户分组下的可用列表，无用户登录获取 default 分组下的可用列表。
- header 中有 Authorization 字段 则查询 key 对应的 token 下配置的模型列表。

**返回示例：**

```json
{
  "data": [
    {
      "id": "gpt-4o-mini",
      "object": "model",
      "created": 1626777600,
      "owned_by": "OpenAI",
      "permission": [
        {
          "id": "modelperm-LwHkVFn8AcMItP432fKKDIKJ",
          "object": "model_permission",
          "created": 1626777600,
          "allow_create_engine": true,
          "allow_sampling": true,
          "allow_logprobs": true,
          "allow_search_indices": false,
          "allow_view": true,
          "allow_fine_tuning": false,
          "organization": "*",
          "group": null,
          "is_blocking": false
        }
      ],
      "root": "gpt-4o-mini",
      "parent": null
    }
  ]
}
```

### 返回结果

|状态码 | 状态码含义 | 说明 | 数据模型|
|---|---|---|---|
|200|OK|none|Inline|
