欢迎使用 **AiHubMix** 提供的 AI 模型 API 路由服务！我们基于统一的 OpenAI API 标准，支持 OpenAI 官方所有模型及市面上其他主流大模型（如 Claude、Gemini、DeekSeek、阿里 Qwen 等）。只需在参数中修改`模型名称`，即可轻松切换接入的大模型。

## 多端点支持说明

- **OpenAI + Azure 双端点**  
  - AiHubMix 会同时链接 OpenAI 官方和微软 Azure OpenAI 两个端点，并随机路由请求至 Azure 进行内容审核  
  - 相比 OpenAI 官方端，Azure OpenAI 自带自动内容过滤，所以响应速度可能略慢

## 快速上手

1. **充值**  
   - 进入「充值」页面完成充值，确保账户余额 > 0  
2. **创建 API Key**  
   - 打开「API Keys」页面，创建并复制自己的 Key  
3. **获取中转信息**  
   - 稳定中转接口地址：`https://aihubmix.com`  
   - 中转 API Key：`sk-***`  
   - 备用服务器地址（新功能测试专用）：`https://api.aihubmix.com`  
4. **替换 API 地址**  
   - 如果你是开发者，请参考 [OpenAI 官方文档](https://platform.openai.com)  
   - 开发时，将文档中的 `https://api.openai.com` 替换为 `https://aihubmix.com` 即可  
   - Claude 模型也支持原生 `v1/messages` 接口，详情请参考 [Claude 原生接口调用](/api/Claude%20原生接口调用)

<Tip>
**提示**：在一些第三方软件或平台输入自定义 URL 时，可能需要添加 `/v1` 或 `/v1/chat/completions` 路径
</Tip>
如：
- `https://aihubmix.com`
- `https://aihubmix.com/v1`（这种情况目前是最多的）
- `https://aihubmix.com/v1/chat/completions`
