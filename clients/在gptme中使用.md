# 在gptme中使用
下载地址：https://github.com/ErikBjare/gptme  
项目部署后进入~/.config/gptme/config.toml文件（位于用户主目录下）进行配置。  
配置文件例子：  
```
[env]
OPENAI_API_KEY = "sk-xxx"
MODEL = "local/gpt-4o"
OPENAI_BASE_URL = "https://aihubmix.com/v1"
```
sk-xxx的位置换成输入[本站的Key](https://aihubmix.com/token)
