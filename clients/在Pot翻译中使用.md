# 在Pot翻译中使用
下载链接：https://pot-app.com/
1. 打开Pot翻译后，在服务设置中选择添加内置服务  
![图片](../media/pot1.png)  
2. 点击OpenAI  
![图片](../media/pot2.png) 
3. 将请求地址设置为 
```
https://aihubmix.com
```    
4. API key从[本站的Key](https://aihubmix.com/token)中创建生成。   
![图片](../media/pot3.png)
5. 划到下方保存  
![图片](../media/pot4.png)
