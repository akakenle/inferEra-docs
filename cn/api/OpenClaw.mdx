## 安装

<Note>
  安装前请确保 Node\>=22
</Note>

执行安装指令

```shellscript
npm install -g openclaw@latest
```

### 修改配置文件

打开 `~/.openclaw/openclaw.json` 文件，增加以下配置：

```json
{
"models": {
    "mode": "merge",
    "providers": {
      "aihubmix": {
        "baseUrl": "http://aihubmix.com/v1",
        "apiKey": "AIHUBMIX_API_KEY",
        "api": "openai-completions",
        "models": [
          {
            "id": "claude-sonnet-4-5",
            "name": "claude-sonnet-4-5",
            "reasoning": false,
            "input": [
              "text"
            ],
            "cost": {
              "input": 0,
              "output": 0,
              "cacheRead": 0,
              "cacheWrite": 0
            },
            "contextWindow": 1000000,
            "maxTokens": 64000
          }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "aihubmix/claude-sonnet-4-5"
      },
      "workspace": "your_workspace_path",
      "compaction": {
        "mode": "safeguard"
      },
      "maxConcurrent": 4,
      "subagents": {
        "maxConcurrent": 8
      }
    }
  }
}
```

## 启动

```shellscript
openclaw gateway
```

## 测试

在聊天页面输入 `hello`，如收到回复则说明配置成功。

## 常见问题

### 版本升级

```shellscript
npm update -g openclaw-aihubmix
```

### Gateway 进程被占用

```shellscript
openclaw gateway stop
openclaw gateway
```
