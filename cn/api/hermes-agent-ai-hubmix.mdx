# 快速开始

## Step 1: 安装 Hermes

打开终端，执行以下命令：

### Linux / macOS / WSL2

```text
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

### Android / Termux

```text
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

> **不支持**原生 Windows 系统，请安装[<u>WSL2</u>](https://learn.microsoft.com/en-us/windows/wsl/install)并从 WSL2 中运行 Hermes Agent。

## Step2：选择供应商

安装成功后，选择 Quick setup -\> More providers -\> Custom endpoint。

# 常用指令

代码块

```text
hermes model          # 选择模型供应商
hermes tools          # 配置要启动的工具
hermes gateway setup  # 设置消息平台
hermes config set     # 设置各个配置参数
hermes setup          # 运行完整设置向导，一次性配置所有内容
```

# Hermes 进化仪表盘

为了让 **Hermes Agent** 的自进化过程清晰可见，AIHubMix 发布了配套的“养马”神器——**Hermes 进化仪表盘**。

它将枯燥的 `~/.hermes/` 目录，转化为一棵具备生命张力的“进化树”。

在这套系统里：

✅ **视觉进化**：基于 D3.js 实时观察 Agent 的成长轨迹——技能如新叶绽放，记忆使树干变粗，并可动态追踪技能源码与健康度。

✅ **全局视角**：Token 趋势、模型成本、工具分布全透明。配合活跃度热力图，支持多维度的运行数据分析。

✅ **极致克制**：**纯只读设计**。在不触动核心配置的前提下，直观把控记忆容量与检索优先级，提供跨对话全文搜索的满格掌控感。

这是一场关于“可视化智能”的实验，也是我们献给 Agent 开发者社区的礼物。
