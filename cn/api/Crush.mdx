## 快速配置指引：

### 1️⃣ 安装地址

可从[官方仓库](https://github.com/charmbracelet/crush)获取应用

### 2️⃣ 环境变量配置

启动后填入 Aihubmix 然后输入密钥即可使用，
密钥可以在 Aihubmix [「Keys」页面](https://aihubmix.com/token) 生成。
