## 请求参数

### 请求头（headers）

<ResponseField type="string" required>
  请求身份认证。接口使用 AIHubMix Manage Key 进行身份认证。
</ResponseField>

<ResponseField type="string">
  请求内容类型。此参数必须设置为 application/json。
</ResponseField>

### 路径参数（Path）

<ParamField body="id" type="integer" required>
  KEY ID。
</ParamField>

### 请求体（Request Body）

<ParamField body="name" type="string">
  新 Key 的名称。
</ParamField>

<ParamField body="expired_time" type="integer">
  过期时间（UNIX 时间戳），-1 表示永不过期。
</ParamField>

<ParamField body="unlimited_quota" type="boolean">
  是否无限额。true 表示无限额；false 表示启用限额（需配合 remain_quota）。
</ParamField>

<ParamField body="remain_quota" type="integer">
  限额余量。remain_quota 为：期望限额 \* 500000（例如期望限额 10，则传 5000000）。 当 unlimited_quota 为 true 时建议传 -1。
</ParamField>

<ParamField body="status" type="integer">
  Key 状态。0 为未启用，1 为启用。
</ParamField>

<ParamField body="models" type="text">
  限制当前 Key 可使用的模型（为空或不传表示不限制）。
</ParamField>

<ParamField body="subnet" type="text">
  IP 限制（为空或不传表示不限制）。
</ParamField>
