**接口说明：**
支持 Ideogram AI 绘图接口，文字绘制能力强劲，支持生图（generate）、混合（remix）、局部编辑（edit）和放大（upscale）等。
- 混合：在参考图的基础上，根据提示词进行重绘，生成新的图片。
- 编辑：在参考图的基础上，根据提示词和蒙版进行局部编辑，生成新的图片。
- 放大：将低分辨率的图片放大至高分辨率，会重绘细节（相似度和细节比例可控制）。

**支持的风格：**
- AUTO：默认的自动选定
- GENERAL：通用
- REALISTIC：写实
- DESIGN：设计
- RENDER_3D：3D
- ANIME：动漫

支持通过 AiHubMix 官方接口调用或 [Cherry Studio APP](https://cherry-ai.com/) 使用，注意目前需要打开代理才能生图，后续会支持国内直连，敬请期待。

## Generate

`POST` https://api.aihubmix.com/ideogram/generate
根据给定提示词和可选参数同步生成图像。图像链接的有效期有限；如果你想保留图像，必须下载保存。

### Require

该接口接受一个对象

**image_request object 必须**

`prompt` string 必须
用于生图的提示词

`aspect_ratio` enum 可选
（不能与分辨率参数同时使用）用于图像生成的宽高比，决定了图像的分辨率。默认为 ASPECT_1_1。

可用比例：
- ASPECT_1_1
- ASPECT_3_1
- ASPECT_1_3
- ASPECT_3_2
- ASPECT_2_3
- ASPECT_4_3
- ASPECT_3_4
- ASPECT_16_9
- ASPECT_9_16
- SPECT_16_10
- ASPECT_10_16

`model` enum 可选 Defaults to `V_2`
用于生成或编辑图像的模型。/generate 和 /remix 支持所有模型类型，但 /edit 仅支持 `V_2` 和 `V_2_TURBO`。

可用模型版本：
- V_1
- V_1_TURBO
- V_2
- V_2_TURBO
- V_2A
- V_2A_TURBO

`magic_prompt_option` enum 可选
提示词增强，可用参数：`AUTO`, `ON`, `OFF`，默认 AUTO（即自动）

`seed` integer 可选`>=0<=2147483647`

`style_type` enum 可选
生成图像时使用的风格类型；此参数仅适用于 `V_2` 及更高版本的模型，不应在 `V_1` 版本中指定。默认 `AUTO`

可用风格：
- AUTO
- GENERAL
- REALISTIC
- DESIGN
- RENDER_3D
- ANIME

`negative_prompt` string 可选
描述不想在图像中出现的内容。仅适用于模型版本 `V_1`、`V_1_TURBO`、`V_2` 和 `V_2_TURBO`。提示词中的描述优先级高于负面提示词中的描述。

`num_images`
单次出图数量，范围 1-8，默认 `1`

## Edit

`POST` https://api.aihubmix.com/ideogram/edit

使用提供的蒙版同步编辑指定图像。蒙版标示出应被编辑的图像部分，而提示词和所选风格类型可进一步引导编辑方向。支持的图像格式包括 JPEG、PNG 和 WebP。图像链接的有效期有限；如果你想保留图像，必须下载保存。

### Request

该接口需要一个包含多个文件的 multipart 表单。

image_filefile 必须

maskfile 必须
- 遮罩图片必须仅包含黑色和白色像素，支持 RGB、RGBA 或灰度图像格式。
- 遮罩图片必须和原图尺寸完全一致。黑色区域表示需要修改的部分，白色区域表示需要保持不变的部分。
- 遮罩不能是纯白色
- 如果修改区域（黑色部分）太小，可能无法完全按照提示词的要求进行修改
- 建议修改区域面积至少占据图片的 10% 以上

prompt string 必须
用于局部编辑的提示词。

`model` enum 必须
用于生成或编辑图像的模型。/generate 和 /remix 支持所有模型类型，但 /edit 仅支持 V_2 和 V_2_TURBO。

可用模型版本：
- V_2
- V_2_TURBO

`magic_prompt_option` enum 可选
提示词增强，可用参数：`AUTO`, `ON`, `OFF`，默认 AUTO（即自动）

`num_images` integer 可选`>=1<=8` 默认 `1`

`seed` integer 可选`>=0<=2147483647`

`style_type` enum 可选
生成图像时使用的风格类型；此参数仅适用于 `V_2` 及更高版本的模型，不应在 `V_1` 版本中指定。

可用风格：
- AUTO
- GENERAL
- REALISTIC
- DESIGN
- RENDER_3D
- ANIME

## Remix

`POST` ****https://api.aihubmix.com/ideogram/remix

根据给定提示词和可选参数来融合提供的图像。输入图像会在重混前裁剪至所选宽高比。支持的图像格式包括 JPEG、PNG 和 WebP。图像链接的有效期有限；如果你想保留图像，必须下载保存。

### Request

根据源图像和提供的描述生成新图像的请求，提供的图像会被裁剪以匹配所选的输出宽高比

**`image_request` object 必须**
使用提供的图像和提示词生成新图像的请求。

`prompt` string 必须
用于生成图像的提示词。

`aspect_ratio` enum 可选
（不能与分辨率参数同时使用）用于图像生成的宽高比，决定了图像的分辨率。默认为 ASPECT_1_1。

可用比例：
- ASPECT_1_1
- ASPECT_3_1
- ASPECT_1_3
- ASPECT_3_2
- ASPECT_2_3
- ASPECT_4_3
- ASPECT_3_4
- ASPECT_16_9
- ASPECT_9_16
- SPECT_16_10
- ASPECT_10_16


`image_weight` integer 可选`>=1<=100` 默认 `50`
参考图权重，默认 50%

`magic_prompt_option` enum 可选
提示词增强，可用参数：`AUTO`, `ON`, `OFF`，默认 AUTO

`model` enum 可选 默认为 `V_2`
用于生成或编辑图像的模型。/generate 和 /remix 支持所有模型类型，但 /edit 仅支持 V_2 和 V_2_TURBO。

`negative_prompt` string 可选
仅适用于 V_1、V_1_TURBO、V_2 和 V_2_TURBO 模型版本。用于描述要从图像中排除的内容。提示词中的描述优先于负面提示词中的描述。

`magic_prompt_option` enum 可选
提示词增强，可用参数：`AUTO`, `ON`, `OFF`，默认 AUTO

`num_images` integer 可选`>=1<=8`Defaults to `1`

seed integer 可选`>=0<=2147483647`

`seed` integer 可选`>=0<=2147483647`

`style_type` enum 可选
生成图像的风格类型；仅适用于 `V_2` 及以上版本的模型，不应在 `V_1` 版本中指定。

## Upscale

`POST` ****https://api.aihubmix.com/ideogram/upscale

使用可选提示词同步放大提供的图像（超分）。支持的图像格式包括 JPEG、PNG 和 WebP。图像链接的有效期有限；如果你想保留图像，必须下载保存。

### Request

该接口需要一个包含文件的 multipart 表单。

**`image_reques` object 必须**
请求对象，用于使用可选提示词放大提供的图像。

`prompt` string 可选
可选的提示词，用于引导放大过程

`resemblance` integer 可选 `>=1<=100` 默认 `50`
相似度

`detail` integer 可选`>=1<=100` 默认 `50`
细节

`magic_prompt_option` enum 可选
提示词增强，可用参数：`AUTO`, `ON`, `OFF`，默认 AUTO

`num_images` integer 可选`>=1<=8` 默认 `1`
生成数量

`seed` integer 可选`>=0<=2147483647`
随机种子

`image_file` file 必须
