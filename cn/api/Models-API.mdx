## 新版本接口

### 获取模型信息平台

**接口地址**：`GET https://aihubmix.com/api/v1/models`

**功能描述**：获取所有可用模型的详细信息，支持多种筛选和排序方式。

#### 接口特性

- **公开接口**：无需身份认证即可访问
- **实时数据**：返回最新的模型配置信息
- **灵活筛选**：支持按类型、标签、功能等多维度筛选
- **智能排序**：支持多种排序策略
- **缓存优化**：支持HTTP缓存机制，提升访问效率
