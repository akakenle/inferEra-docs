### 官网

N8N 官网：http://n8n.io/

### 配置教程

1. 进入工作区，新增 Chat Model 节点，增加凭据。

![n8n](/public/cn/n8n-1.jpg)

2. 填写名称、API Key 以及 Base URL 并保存。

![n8n](/public/cn/n8n-2.png)

3. 选择新建的凭据，添加需要使用的模型，并关闭 `Use Response API` 。

![n8n](/public/cn/n8n-3.png)

4. 打开 `Open Chat` 即可在 N8N 中使用 AIHubMix 的模型。

![n8n](/public/cn/n8n-4.jpg)

![n8n](/public/cn/n8n-5.png)
