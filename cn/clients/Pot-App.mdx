1. 打开 [Pot 翻译](https://pot-app.com/)后，在服务设置中选择添加内置服务  
![图片](../../public/cn/pot1.png)  
2. 点击 OpenAI  
![图片](../../public/cn/pot2.png) 
3. 将请求地址设置为 
```
https://aihubmix.com
```    
4. API key 从[本站的Key](https://aihubmix.com/token)中创建生成。   
![图片](../../public/cn/pot3.png)
5. 划到下方保存  
![图片](../../public/cn/pot4.png)
