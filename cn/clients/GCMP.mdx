---
title: "在 GCMP 中使用"
---

1. 在 VS Code 扩展市场搜索`GCMP`并安装。

![GCMP-1](/public/cn/fs-1.png)
   
2. 打开 VS Code  的 `GitHub Copilot Chat `面板，在模型选择器的底部选择模型管理。

![GCMP-2](/public/cn/GCMP-2.png)

   
3. 点击`Manage Models` ，选择 provider `OpenAI / Anthropic Compatible` ，创建模型。

![GCMP-3](/public/cn/GCMP-3.png)


4. 填写基本信息，选择提供商`aihubmix `。

![GCMP-4](/public/cn/GCMP-4.png)

5. 管理 API 密钥，并填写 AIHubMix API Key。
   
![GCMP-5](/public/cn/GCMP-5.png)
