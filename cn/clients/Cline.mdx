> Cline 即原来的 Claude Dev

- 下载插件，进入设置页面。  
- API Provider 一栏选择 OpenAl Compatible。  
- Base URL 一栏输入：
```
https://aihubmix.com/v1
```
- API Key 一栏从[本站的 Key](https://aihubmix.com/token)中生成。  
- 输入模型名后保存设置即可使用。
- **注：模型名称建议从本站设置页面复制粘贴。图片示例为 claude-3-5 模型，gpt-4o 也可以使用。**
